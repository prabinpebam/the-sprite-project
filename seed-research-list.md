# The sprite prooject
This project is an atempt at building the ultimate tool for sprite and all sprite related stuff for game asset generation which can also be used in websites.


We are just at initial research. Below are a bunch of sites that I want to gather detailed research on in order to use the research findings for further ideation of how to take the project forward. So, the research has to be done in order to get a good understanding of each project and characterize what it is and what it offers.

This has to be documented with the skills in C:\personal\the-sprite-project\docs-presentation-skill\SKILL.md
