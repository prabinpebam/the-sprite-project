Broader Libraries

Kenney.nl

A huge, public-domain, general-purpose game-asset library - the contrast point to the LPC world and a benchmark for "batteries-included" asset delivery.

TL;DR

Kenney offers thousands of free, largely CC0 game assets - 2D and 3D art, audio, UI - alongside creation tools and genre starter kits. Unlike the collaborative, attribution-based LPC ecosystem, Kenney is a consistent, mostly single-studio house style with no attribution required, funded through an optional club/Patreon and an all-in-one paid bundle.

What it offers

Asset packs

Thousands of drop-in assets across genres, downloadable individually or as one all-in-one package.

Tools

Creation tools that produce 3D models without complex software or frameworks, usable in most engines.

Starter kits

Free, open-source kits covering various game genres to kickstart a new project.

How it differs from LPC

Kenney

  • Finished, ready-to-use packs (art + audio + UI, 2D & 3D)
  • Consistent single-studio house style
  • CC0 - no attribution, no share-alike
  • Tools + starter kits included

LPC ecosystem

  • Modular, layer-composited characters
  • Many authors under one style guide
  • Attribution-based (CC-BY-SA / CC-BY / OGA-BY / GPL, some CC0)
  • Generators + engine plugins around a shared corpus

Relevance to this project

Kenney is the north star for frictionless asset consumption: massive breadth, zero licensing overhead, and companion tools/kits. Where LPC wins on modular character depth, Kenney wins on ease-of-use - a useful tension when deciding this project's positioning.
