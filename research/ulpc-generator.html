Character Generators

Universal LPC Spritesheet Generator

The flagship, community-maintained web app that composes LPC character art into ready-to-use spritesheets - and the reference the whole ecosystem points back to.

TL;DR

Maintained under the LiberatedPixelCup organization, this browser-based generator attempts to include all LPC character art to date. You pick body, hair, clothing, and gear layers; it live-previews animations, recolors via GPU shaders, and exports a spritesheet plus a precise credits file. At ~1,400 stars and 522 forks it is by far the most adopted project in this research. The Gaurav0 repo is an older fork of the same codebase.

What it offers

Layered composition

Body, head, hair, headwear, arms, torso, legs, feet, tools, and weapons - selected and stacked into a single character. Body types: Male, Female, Teen, Child, Muscular, Pregnant.

Real-time recoloring

GPU-accelerated WebGL palette swapping (fragment shaders, 1-32 colors per variant) with an automatic CPU fallback - roughly 2x faster than CPU per preview.

Animation preview

Live animation playback with documented frame cycles, plus a full spritesheet preview and zoom.

Export & credits

Download a spritesheet PNG, ZIP splits (by animation, item, or frame), and CREDITS as CSV/TXT. Export/Import selections as JSON to restore a build later.

License filtering

Filter available items by license and by animation, so you can constrain a character to only the terms your project can accept.

Engine plugins

Community plugins for Godot 3.5 & 4.2, plus an RPG Maker MZ starter kit under the same organization.

Export pipeline

  1. Select layers

    Choose body type and stack items across categories; auto-match heads/ears to body color.

  2. Preview & recolor

    Watch animations live; recolor palettes in real time via WebGL.

  3. Export

    Download the spritesheet (PNG), split ZIPs, and the matching CREDITS file - or export selections to JSON.

  4. Attribute

    Ship the generated credits so every author of every used layer is properly acknowledged.

Relevance to this project

This is the de-facto standard for LPC character assembly. Any new sprite tool will inevitably be compared to it - its layer model, license-aware exports, and WebGL recoloring are the baseline feature set to match or improve upon.
