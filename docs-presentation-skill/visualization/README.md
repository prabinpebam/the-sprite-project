# Visualization Pipeline

One skill owns the complete visual workflow:

[`svg-illustration`](./skills/svg-illustration/SKILL.md) identifies visual opportunities, selects
and designs the right representation, creates deterministic accessible SVG, applies customizable
light/dark themes and visual personalities, adds tasteful optional motion, and validates the real
render.

## Progressive Layout

```
skills/
  svg-illustration/
    SKILL.md          # entry point (frontmatter: name, description)
    references/       # load only the topic needed
    assets/           # theme/preset templates
    scripts/          # SVG and theme validators
```

The package intentionally contains no library-specific chart, graph, infographic, or animation
skill. Their useful visualization principles are synthesized into focused references while final
ownership remains singular.
