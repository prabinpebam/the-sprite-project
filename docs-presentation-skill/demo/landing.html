Slate

A portable, zero-configuration viewer that turns Markdown and HTML content into a beautiful, navigable documentation site.

TL;DR

This demo is a working content root. It renders HTML and Markdown pages through one pipeline, styles them with a token-driven design system, and adds navigation, search, a table of contents, theming, and deep links - with no build step.

How it works

You provide

  • Content pages (.html or .md)
  • A docs-manifest.json
  • Optional slate.config.json for branding

Slate provides

  • Design system + components
  • Navigation, search, TOC, deep links
  • Theming, sanitization, responsiveness

Tip

Toggle the theme (top right), press Ctrl/Cmd-K to search, and edit slate.config.json's brandColor to rebrand the whole site with no code change.
