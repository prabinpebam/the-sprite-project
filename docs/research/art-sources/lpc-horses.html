LPC Horses & the Stendhal Rework

Animal sprites in LPC style - the original animated horse set and a later rework that added mythical variants and optimized files.

TL;DR

Two related OpenGameArt submissions. The original [LPC] Horses by bluecarrot16 (2016) delivers fully animated horses - walk, eat, and gallop in four directions, plus an 11-frame gallop and five color recolors. The LPC Horses Rework by AntumDeluge re-cut it for the Stendhal game with indexed-color PNGs and added a unicorn and pegasus. Both are broadly licensed (CC-BY 3.0 / GPL 2.0 & 3.0 / OGA-BY 3.0).

Original vs. rework

[LPC] Horses (bluecarrot16)

  • Richer animation set (walk/eat/gallop + 11-frame gallop)
  • 5 realistic recolors
  • Spawned further reworks (BenCreating's extended horse, riding kits)

Horses Rework (AntumDeluge)

  • Optimized indexed-color PNGs for a shipping game
  • Adds mythical unicorn & pegasus
  • Tighter frame set (3 per direction) tuned for Stendhal

Relevance to this project

Horses show LPC beyond humanoids, and the original-plus-rework pair is a live example of the open-asset remix loop: one artist's animated base gets re-cut, optimized, and extended by others for real games - exactly the kind of derivative pipeline a sprite platform must support.
