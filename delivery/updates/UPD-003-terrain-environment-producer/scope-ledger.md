# UPD-003 Scope Ledger

## Included

- one optional terrain document in the existing project;
- four built-in procedural materials;
- 16 cardinal 32px autotiles;
- fixed 12×8 paintable preview;
- terrain-local palette derived from project theme;
- save/reopen/conflict/snapshot/copy/offline;
- archive-v3 web/Electron portability;
- generic and Godot terrain packages with credits.

## Deliberately Excluded

See the canonical `OUT-UPD003-*` records in `traceability.json`: terrain packs, transitions, 47-tile/Wang profiles, animated water, decorators, arbitrary/imported maps, scenes, structures/props, lighting maps, global token redesign, new engines, and pixel editing.

Anything plausible but absent from both the positive traceability graph and exclusion ledger is a specification defect.
