# UPD-004 Scope Ledger

## Included

- 1–16 named character recipes per project;
- create from active pack defaults;
- switch active character;
- duplicate, rename, and delete with recovery;
- separate recipe records/files under one atomic project save;
- active-only compose, overrides, preview, credits, and export;
- web, Electron, archive, terrain, offline, and cross-host compatibility.

## Excluded

Folders/tags/search/reorder, batch export, scene placement, per-character project themes, recipe import/merge, collaboration/cloud, drag-and-drop, and more than 16 characters are explicit `OUT-UPD004-*` records in `traceability.json`.
