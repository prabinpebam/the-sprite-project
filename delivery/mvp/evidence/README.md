# MVP Evidence

Actual-UI runs are stored under `runs/<RUN-ID>/`. Each run records the commit, production build, browser environment, raw screenshots and captures, downloads, console/page/network events, expectation outcomes, and anomalies.

## Acceptance

The MVP requires every `TF-*` and `EB-*` to have a passing latest run and no unresolved severity 0-2 issue. Unit and integration checks remain supporting evidence only.
