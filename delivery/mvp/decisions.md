# MVP Decisions

## DEC-STACK

- **Decision:** Use a local-first React and TypeScript Vite application with Canvas 2D, JSON contracts, Vitest, and Playwright.
- **Reversible:** Yes
- **Evidence:** Browser rendering, local persistence, ZIP export, and actual-UI automation are all supported without a service.
- **Status:** verified

## DEC-CONTENT

- **Decision:** Bundle two original CC0-compatible demonstration packs with complete project-authored provenance, while retaining the manifest contract needed for later reviewed LPC corpus intake.
- **Reversible:** Yes
- **Evidence:** Eliminates unresolved third-party legal approval while proving composition, tokens, packs, attribution, and export end to end.
- **Status:** verified

## DEC-GODOT

- **Decision:** Target Godot 4 text SpriteFrames resources in the MVP package and validate their declarations against the generic animation manifest.
- **Reversible:** Yes
- **Evidence:** Text resources are diffable and require no editor plugin or manual slicing.
- **Status:** verified
